# 📊 Sales & Revenue Analysis Dashboard

🚀 An interactive Sales & Revenue Analysis Dashboard built using Power BI to analyze sales performance, track KPIs, and generate business insights from the Superstore dataset.

---

# 🎯 Project Objectives

✅ Analyze sales and profit performance  
✅ Track important KPIs  
✅ Identify top-performing regions and categories  
✅ Build interactive visualizations using slicers and charts  

---

# 🛠️ Tools Used

- Power BI
- Microsoft Excel
- DAX
- Power Query

---

# 📊 Dashboard Features

📌 KPI Cards:
- 💰 Total Sales
- 📈 Total Profit
- 📦 Number of Orders
- 🛒 Products Sold

📌 Visualizations:
- 📈 Monthly Sales Trend
- 🌍 Region-wise Sales Analysis
- 🛍️ Category-wise Sales
- 👥 Customer Segment Analysis

📌 Interactive Features:
- 🎛️ Region & Category Slicers
- 🔍 Dynamic Filtering

---

# 📈 Key Insights

✅ Technology category generated highest sales  
✅ West region contributed maximum revenue  
✅ Consumer segment had highest sales contribution  
✅ Sales trends showed monthly fluctuations
